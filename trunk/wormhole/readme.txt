Introduction
--------------

WormHole is a VPN suite. It use libpcap capture the packet you special, and send these packet to a WomrHole router. In the other side, WormHole receive these packet and send to ethernet.

The WormHole router can be many types of channel: TCP, ssh, http, xmpp, and so on. So WormHole aid work anywhere with any type of internet connections.
